# Day 6: Switches vs. Routers

Today I studied the hardware that makes networking possible.

### Key Differences:
- **Switches (Layer 2):** Move data within a local network using MAC addresses.
- **Routers (Layer 3):** Move data between different networks using IP addresses.

### Professional Insight:
If I am investigating an attack *inside* an office, I look at Switch logs. If the attack is coming from *outside*, I look at Router and Firewall logs.

### Active Recall:
- **Q:** Which device uses MAC addresses to deliver data?
- **A:** A Switch (Layer 2).
