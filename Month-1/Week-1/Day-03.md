# Day 3: Common Network Protocols and Gateways

Computers use specific "languages" (protocols) and enter through specific "doors" (ports).

### Important Ports to Monitor:
- Port 22 (SSH): Used for secure remote management.
- Port 53 (DNS): The internet's directory service.
- Port 443 (HTTPS): Secured web traffic.

### SOC Perspective:
Monitoring for traffic on unusual ports is a primary way we detect unauthorized access or "backdoors" in a network.

### Active Recall:
- Q: Why is HTTP (Port 80) considered a security risk?
- A: It is unencrypted, meaning data is sent in plain text and can be intercepted.
